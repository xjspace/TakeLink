# Refactor Plan - LAN Happy

## 项目概述

**目标**: 手机远程控制电脑上的 Claude Code IDE

**技术栈**:
- cli/: Node.js + Express + Socket.io + node-pty
- app/: React Native (Expo) - 目前未使用
- Web UI: xterm.js 终端模拟

2. app/ 目录中有未使用的消息模式组件
3. 前端 JS 代码结构不清晰

1. 单文件过大 (index.ts 350+ 行)
2. 前端 JS 内嵌在 HTML 中
3. 缺少类型定义共享
